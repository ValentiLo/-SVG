Прозрачность обводки

Прозрачность задаётся свойством stroke-opacityсо значениями от 0 до 1, например: stroke-opacity="0.5".

либо через CSS:

rect {

stroke: orange;

stroke-opacity: 0.5;

}

Задачи

  • С помощью атрибута задайте первой фигуре прозрачность обводки со значением0.6,

  • а второй фигуре — со значениемstroke-width: 5%.

  • В CSS третьей фигуре задайте прозрачность обводки 0.5.

Было

object.svg

        <svg width="300" height="150">
        <circle r="50" cx="75" cy="75" fill="none" stroke="darkorange" stroke-width="40"></circle>
        <circle r="50" cx="150" cy="75" fill="none" stroke="gold" stroke-width="40"></circle>
        <circle class="shape-stroke" r="50" cx="225" cy="75" fill="none" stroke="lightseagreen" stroke-width="40"></circle>
        </svg>

style.css

svg { border: 1px solid #dddddd; }

Надо

Код

<svg width="300" height="150">

<circle r="50" cx="75" cy="75" fill="none" stroke="darkorange" stroke-width="40" stroke-opacity="0.7"></circle>

<circle r="50" cx="150" cy="75" fill="none" stroke="gold" stroke-width="40" stroke-opacity="0.6"></circle>

<circle class="shape-stroke" r="50" cx="225" cy="75" fill="none" stroke="lightseagreen" stroke-width="40"></circle>

</svg>

style.css

svg { border: 1px solid #dddddd; }

.shape-stroke {

stroke-opacity: 0.5;

}
