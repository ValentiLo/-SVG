Отсутствие заливки

Иногда бывает нужно полностью убрать заливку, например, если вам нужен только контур фигуры. Это можно сделать ключевым словом none, результатом будет полная прозрачность фигуры.

Задачи

  • Задайте первой фигуре отсутствие заливки с помощью fill="none".

  • Сделайте то же самое для третьей фигуры, но через CSS, используя fill: none.

Было

object.svg

        <svg width="300" height="150">

        <circle r="60" cx="75" cy="50%">
          
        <circle r="60" cx="150" cy="50%" fill="hsla(50, 100%, 50%, 0.75)">
          
        <circle class="shape-transparent" r="60" cx="225" cy="50%">
          
        </svg>

style.css

svg { border: 1px solid #dddddd; }

.shape-transparent { }

Надо

Код

<svg width="300" height="150">

<circle r="60" cx="75" cy="50%" fill="none"></circle>

<circle r="60" cx="150" cy="50%" fill="hsla(50, 100%, 50%, 0.75)"></circle>

<circle class="shape-transparent" r="60" cx="225" cy="50%"></circle>

</svg>
